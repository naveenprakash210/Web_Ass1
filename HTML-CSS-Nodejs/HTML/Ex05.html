How to create Lists in HTML?

Lists are available as ordered and Unordered Lists. UnOrdered Lists are bulletted Lists and Ordered Lists are lists with numberings. numberings could be either Indo-Arabic Numbers or Roman numbers. U can also have alpabets. The type is defined using type attribute in the lists.
